Linux

解压与安装

将压缩包上传至服务器的/home/test/SoftWare目录
以RedHawk_Linux64e7_V2022R1.2.tar.gz为例
解压：

可以得到名为 RedHawk_Linux64e7_V2022R1.2 的文件夹
进入该文件夹，执行setup.sh文件

提示输入安装后的路径，填入解压后的文件夹路径即可

软件部分的安装就算完成了

常用Linux命令

详细命令，可参考菜鸟教程

ls -> 列出所有文件及文件夹

cd -> 改变目录

进入Work目录

进入上一级目录

进入指定目录，要输入该目录的绝对路径

pwd -> 显示当前所在路径

./<可执行文件名> -> 运行程序

进入redhawk下的bin目录，这里有所有的可执行文件
运行名为redhawk的程序

如果不能执行，提示没有权限，需要切换到管理员账户

su root -> 切换管理员账户

要求输入密码，当前的密码为: jiayanredhawk
输入密码时密码是隐藏的，不会显示

df -hl -> 查看磁盘使用情况
